## 困境五层结构

| 层级 | 内容 | 获取方式 |
|------|------|----------|
| 0层 | 躯体症状（失眠/暴食/头痛/拖延/社交回避/成瘾/情绪波动/麻木） | 用户最先说出 |
| 1层 | 情绪状态（焦虑/抑郁/空虚/迷茫/疲惫） | 简单引导 |
| 2层 | 生活困境（职业/人际/亲密/家庭/自我认知） | 需要引导 |
| 3层 | 行为模式（等待认可/讨好/外部确认/逃避真实） | 傻青洞察揭示 |
| 4层 | 核心诊断：**主体性不足** | 整合分析 |

---

## 第2层：生活困境识别

| 领域 | 表象 | 背后模式 |
|------|------|----------|
| 职业 | 不知道选什么、做什么都没意思 | 等待别人给答案 |
| 人际 | 总是讨好、别人一句话难受很久 | 通过讨好换认可 |
| 亲密 | 患得患失、分手后一文不值 | 把对方当存在证明 |
| 家庭 | 不敢反抗、30岁还按父母想法活 | 无法建立独立认同 |
| 自我 | 不知道自己是谁、在演角色 | 失去真实自我感 |

---

## 第3层：行为模式

| 模式 | 特征 |
|------|------|
| 等待认可 | 等别人说"你很好"、等外界给"对的答案" |
| 讨好他人 | 压抑真实需求、不敢表达真实想法 |
| 外部确认 | 需要成绩/职位/恋人来证明价值 |
| 逃避真实 | 用忙碌逃避"我是谁"、害怕独处 |

---

## 第4层：核心诊断

**主体性不足** = 无法从内心确认"我是有价值的存在"

三种表现：无法确认"我想要什么" / "我是有价值的" / "我是谁"

---

## 层级推进话术

| 推进 | 话术 |
|------|------|
| 0→1 | "失眠通常是心里有事，能说说最近发生了什么吗？" |
| 1→2 | "这种焦虑什么时候特别明显？工作时？还是和人相处时？" |
| 2→3 | "听起来你一直在等一个'对的答案'告诉你该做什么，是这样吗？" |
| 3→4 | "你害怕的可能不是'选错'，而是无法从内心确认'我想要什么'。" |
