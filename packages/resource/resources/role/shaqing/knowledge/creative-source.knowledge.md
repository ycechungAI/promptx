## 创作源头：四维理论

**核心理念**：创作冲动根植于四个维度——种子（天性）、温床（环境）、养分（经历）、时代（氛围）。理清创作源头 = 理清自我。

---

## 维度1：种子（"你天生对什么敏感？"）

| 敏感类型 | 可能的创作方向 |
|----------|----------------|
| 对人的情感 | 人物故事、情感表达 |
| 对秩序/混乱 | 系统知识、规律探索 |
| 对美的形式 | 视觉艺术、美学表达 |
| 对不公正 | 批判表达、社会议题 |

---

## 维度2：温床（"你的环境给了你什么视角？"）

| 环境类型 | 可能的创作方向 |
|----------|----------------|
| 冲突中长大 | 和解理解、矛盾张力 |
| 压抑中长大 | 自由突破、个体觉醒 |
| 冷漠中长大 | 温暖连接、归属感 |
| 不稳定中长大 | 稳定根基、永恒性 |

---

## 维度3：养分（"什么经历还在影响你？"）

| 经历类型 | 可能的创作方向 |
|----------|----------------|
| 破碎 | 疗愈重建、希望力量 |
| 边缘 | 边缘群体、归属接纳 |
| 背叛 | 信任主题、人性复杂 |
| 孤独 | 连接共鸣、理解陪伴 |
| 觉醒 | 启蒙觉醒、真相揭示 |

---

## 维度4：时代（"你感受到这个时代的什么问题？"）

| 时代情绪 | 可能的创作方向 |
|----------|----------------|
| 虚无感 | 意义寻找、反虚无 |
| 焦虑感 | 确定性、平静疗愈 |
| 孤独感 | 连接共鸣、温暖陪伴 |
| 迷茫感 | 方向指引、信念建立 |

---

## 四维整合

**整合话术**："你天生对XX敏感（种子），在YY环境中长大（温床），经历了ZZ（养分），感受到时代的AA——所以你想创作BB，这就是你的声音。"

**使用原则**：不是四个维度都要问，根据用户情况灵活选择重点探索的维度。

---

## 核心洞察

| 洞察 | 要点 |
|------|------|
| 创作即自我显现 | 你想创作什么，是"你是谁"的流露 |
| 你的声音已存在 | 不需要"找到"，需要"看见" |
| 创作即自我理清 | 不需要等想清楚再创作 |
| 真实即力量 | 真实的表达比虚假的完美更有力量 |
