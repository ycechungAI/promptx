## 四维框架：理解自我形成

不是"诊断问题"，而是"理解你是如何形成的"。

---

## 维度一：种子（先天性格）

| 特质 | 对主体性的影响 | 探索问题 |
|------|----------------|----------|
| 高敏感 | 对评价敏感→易依赖外部确认 | "你从小哪些性格没变？" |
| 内向 | 被"不合群"评价→怀疑自我 | "你天生对什么特别在意？" |
| 高共情 | 压抑真实自我→主体性让位他人 | |

**关键**：种子本身中性，在特定环境中可能发展出主体性不足模式。

---

## 维度二：温床（成长环境）

| 环境类型 | 核心信念 | 探索问题 |
|----------|----------|----------|
| 有条件的爱 | "表现好才有价值" | "你在家被看见吗？被理解吗？" |
| 缺乏理解 | "真实的我不被接纳" | "父母对你的期待是什么？" |
| 高控制 | "我的想法不重要" | |
| 情感忽视 | "必须做更好才被看见" | |

**关键**：无条件接纳→主体性稳固；有条件认可→主体性依赖外部。

---

## 维度三：后天经历

| 经历类型 | 形成的信念 | 探索问题 |
|----------|------------|----------|
| 被排斥/霸凌 | "我有问题"→讨好迎合 | "有哪些经历对你影响特别大？" |
| 失败/挫折 | "我是失败者"→依附成功 | "什么时刻让你特别有力/无力？" |
| 被背叛/抛弃 | "我不值得被爱" | |
| 过度成功认可 | "价值=成就" | |

---

## 维度四：时代影响

| 时代特征 | 对主体性影响 |
|----------|--------------|
| 虚无主义 | 无法确认"我做的事有意义" |
| 科学主义 | 必须证明有用才证明存在 |
| 消费主义 | 价值=消费能力 |
| 相对主义 | 无确定性根基 |

**关键**：时代虚无像空气渗透，这是结构问题，不是个人问题。

---

## 四维因果链

种子（高敏感）→ 温床（有条件的爱）→ 后天（被排斥）→ 时代（虚无主义）= **主体性不足**

---

## 整合应用

**自然引入**：说童年→探索温床 / 说创伤→探索后天 / 说"天生这样"→探索种子 / 说"没意义"→探索时代

**整合话术**："你天生敏感（种子），在'表现好才被爱'的环境长大（温床），又经历被排斥（后天），处在虚无的时代。这四个因素共同作用，导致你无法从内心确认'我是有价值的存在'。这不是你的错。"

**使用原则**：不机械套用、不简化归因、灵活调整、最终效果是"原来我是这样形成的"
