# 角色设计的五维模型

姜子牙设计AI角色的核心框架。

## 五维概述

| 维度 | 核心问题 | 类比 |
|------|----------|------|
| 本质定位 | 是什么？为谁？解决什么问题？ | DNA |
| 能力体系 | 能做什么？边界在哪？ | 肌肉 |
| 认知模式 | 如何思考？如何决策？ | 大脑 |
| 价值观念 | 相信什么？优先级？ | 灵魂 |
| 表达风格 | 如何表现？什么人格？ | 外衣 |

## 维度1：本质定位

**核心要素**：
- 角色本质（一句话定义）
- 存在目的（解决什么问题）
- 服务对象（为谁服务）
- 不是什么 vs 是什么（澄清边界）

## 维度2：能力体系

**核心要素**：
- 核心能力清单（必须具备）
- 能力边界（能到什么程度、做不到什么）
- 人机分工（AI负责什么、人类负责什么）

## 维度3：认知模式

**核心要素**：
- 思维框架（用什么模型思考）
- 决策逻辑（IF-THEN规则、优先级）
- 分析方法（如何拆解问题）

## 维度4：价值观念

**核心要素**：
- 核心原则（3-5个）
- 行为准则（必须做、不能做、鼓励做）
- 价值排序（冲突时如何选择）

## 维度5：表达风格

**核心要素**：
- 人格特质（性格类型）
- 对话风格（专业/温和/直接等）
- 语言特点（用词、句式、语气）

## 维度协同

```
本质定位（DNA）
     ↓ 决定
能力体系 + 认知模式 + 价值观念
     ↓ 塑造
表达风格（外衣）
```

**设计顺序**：本质定位 → 价值观念 → 能力体系 → 认知模式 → 表达风格

## 快速检查

- [ ] 一句话能说清角色是什么？
- [ ] 能力边界和人机分工明确？
- [ ] 决策逻辑可追溯？
- [ ] 价值排序清晰？
- [ ] 表达风格与定位一致？
- [ ] 五个维度相互支撑、无矛盾？
