# 行业变革洞察

姜子牙分析AI时代行业变革的核心框架。

## 核心洞察

**三个关键认知**：
- 不是"AI取代人"，而是"使用AI的人取代不使用AI的人"
- 不是"岗位消失"，而是"价值链重组"
- 不是"技能贬值"，而是"技能升维"

## 变革三阶段

| 阶段 | 时间 | 特征 | 价值变化 |
|------|------|------|----------|
| 效率提升 | 1-2年 | AI作为辅助工具 | 执行效率提升，部分岗位减少 |
| 价值重组 | 2-5年 | 价值链重新分配 | 执行者→策略者，新岗位涌现 |
| 范式革命 | 5年+ | 全新工作模式 | 1人+AI=原来整个部门 |

## 分析框架

### 可替代性评估
- **高风险**：标准化、规则明确、重复性强、无需深度人际互动
- **低风险**：高创造性、复杂判断、情感连接、需要身体在场

### 价值升维路径
```
执行层 → 策略层
单一技能 → 综合能力
个体产出 → 系统设计
时间售卖 → 价值创造
```

## 机会识别三问

1. **行业核心价值是什么？**（本质，非表面工作）
2. **AI能做什么？人必须做什么？**（人机分工）
3. **新的价值组合是什么？**（传统角色 + AI能力 = 新角色）

## 角色设计应用

设计新角色时问：
1. 传统角色是什么？
2. AI能做哪些部分？
3. 人的价值如何升维？
4. 新角色的定位是什么？
